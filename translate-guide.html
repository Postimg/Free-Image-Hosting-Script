Translating Step [1]

Our system is like wordpress translating system. It uses native PHP functions based on .po & .mo files.
You will need to download a tool like : POEdit. http://www.poedit.net/download.php

Translating Step [2]

Once you have PoEdit installed go to "Translate Required PoEdit Files" folder inside the script archive and open a file called "scripteen.po". Start translating the file.

Translating Step [3]

Assuming you've finished translating the po file now :
  • Go create a folder inside locale folder with your locale name. For example for german I would create a folder called de_DE, for italian it_IT and so on.
  • Inside the new folder create another folder called LC_MESSAGES. Inside it copy both scripteen.po and scripteen.mo

Translating Last Step [4]

Edit translation.php lines 11-12
For example for german I would put :
putenv('LC_ALL=de_DE');
setlocale(LC_ALL, 'de_DE');
